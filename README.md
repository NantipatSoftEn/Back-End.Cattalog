## API Cattalog

### Requirement
Use MongoDB and    [express-resource](https://github.com/expressjs/express-resource)

## Pattern API
```
GET     /c              ->  index
GET     /c/new          ->  new
POST    /c              ->  create
GET     /c/:c           ->  show
GET     /c/:c/edit      ->  edit
PUT     /c/:c           ->  update
DELETE  /c/:c           ->  destroy
```

## How To use
```
  open mongod then
  npm start
```
